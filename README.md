## App Title: Text Correction Tool

**Description:** A web application that utilizes OpenAI's API to correct user-provided text, highlighting errors and suggested corrections.

**Structure:**
```plaintext
src/
├── app/
│   ├── api/
│   │   └── corregir/
│   │       └── route.ts
│   └── page.tsx            # Main application page
├── components/
│   ├── ui/
│   │   ├── TextInput.tsx
│   │   ├── Modal.tsx
│   │   └── Loading.tsx
├── types/
│   └── index.d.ts
└── utils/
    └── formatResponse.ts
```

**Technology:**
- Next.js 15
- TypeScript
- OpenAI API
